HTML Entities

HTML entities are special characters that can be used in HTML. For example, to display an ampersand (&), you would use the entity &amp;. Here are a few more examples:

  • Less than: &lt; = <
  • Greater than: &gt; = >
  • Copyright symbol: &copy; = ©
  • Trademark symbol: &trade; = ™
